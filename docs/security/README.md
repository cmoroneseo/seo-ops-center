# Security Workflow Index

Start here when reviewing or improving security for SEO Ops Center.

- [DevSecOps workflow](devsecops-workflow.md): operating system for development, review, deployment, and maintenance.
- [Codex prompts](codex-prompts.md): reusable prompts for audits, PR reviews, RLS reviews, endpoint reviews, and AI-generated code safety checks.
- [Checklists](checklists.md): GitHub, Vercel, Supabase, API, PR, and AI coding checklists.
- [Scheduled workflow](scheduled-security-workflow.md): daily, weekly, monthly, and quarterly review cadence.
- [Privacy data map](privacy-data-map.md): data inventory, access rules, AI data rules, and minimization defaults.
- [Triage](triage.md): severity definitions, fix targets, and review expectations.

Local commands:

```bash
npm run security:all
npm run security:env:production
npm run security:static:strict
```
